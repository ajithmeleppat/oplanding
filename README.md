This is my attempt at the landing page assignement from The Odin Project fundamentals. The landing page contains the following:
1. Header - logo on the left and navbar on the right
2. Intro the website, with a signup button and an image
3. Additional information with 4 images and labels for each
4. a quote or testimonial
5. A banner for sign up with a button
6. footer with copyright info
